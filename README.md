#  未授权访问漏洞检测小工具(WxAgenTinfoCheck)

1、支持 AgenTinfo 未授权访问漏洞检测

2、支持泄露的corpid、corpsecret利用，当前仅支持获取部门列表、成员ID信息

3、需要.Net4.0支持
